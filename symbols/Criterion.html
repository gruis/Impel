Class Detail
Criterion(parent, peer_column, value, comparison)
An inner class that describes the different portions of the WHERE clause of an SQL query. The Criterion class should not be directly instantiated. Rather it should be instantiated through Criteria.getNewCriterion 
var c     = new Criteria();
var crit  = new Criterion(c,"CardPeer::name","Queen of Hearts",CritConstants.NOT_EQUAL);
Parameters:
{Criteria} parent
The Criteria object to which this Criterion object is attached
{String} peer_column
The Peer::column to which this Criterion object applies
{mixed} value
The value of the peer_column
{String} comparison Optional, Default: CritConfig.EQUAL
The relationship between the peer_column and the value
See:
